# -Color-Flipper
カラーフリッパー

このプロジェクトでは、ボタンをクリックするたびにランダムな背景色を生成し、画面全体の背景色を動的に変更するウェブアプリケーションを作成します(JavaScript: DOM操作とランダムカラー生成のロジック)。
